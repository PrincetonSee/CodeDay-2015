CipherSpeak: The next generation of Desktop-based encrypted chat.

The basis of the application runs on the concept of an encryption key. Each of these encryption keys allows for a private chat between two or more users, as long as each user has the same key. Once the key is first typed in, lines of "gibberish" will then be translated into understandable text. Without the key, this information is just gibberish.Users are able to group chat on the left side of the chat bar, while the right side lists out the decrypted versions of the texts. Our hope is that this will allow users to connect on both a public and private level.
